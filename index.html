Level 1: Dari Nol sampai jago HTML

Kalian akan belajar HTML dari dasar, mulai dari struktur, cara menulis HTML dan di akhir kalian akan mencoba membuat website sederhana menggunakan HTML.

🚀

Level 2: Power Up CSS

Kalian akan mempelajari dasar CSS bagaimana cara mempercantik tampilan website kalian, mulai dari warna, font, layout, dan responsive design. Di akhir kalian akan membuat website yang lebih menarik dengan CSS.

🧠

Level 3: Brain Boost Javascript

Asumsikan javascript adalah otak dari website kalian, kalian akan belajar logika menggunakan javascript, mulai dari variabel, fungsi, kondisi, DOM Manipulation, dan event handling. Di akhir kalian akan membuat website yang interaktif dengan javascript.

🧠

Level 4: Advance Javascript: Menggunakan React

React adalah library javascript yang sangat populer untuk membangun user interface. Kalian akan belajar bagaimana cara menggunakan React untuk membuat komponen, mengelola state, dan menghubungkan komponen-komponen tersebut. Di akhir kalian akan membuat aplikasi web yang lebih kompleks dengan React.
